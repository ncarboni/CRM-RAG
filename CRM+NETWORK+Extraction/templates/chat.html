About This Chat

This chat interface uses an advanced graph-based Retrieval-Augmented Generation (RAG) system with the following features:

  • Graph-Based Document Retrieval: Documents are organized as a graph with weighted edges based on CIDOC-CRM relationship semantics
  • Coherent Subgraph Extraction: Uses greedy selection to balance individual document relevance with network connectivity
  • Relationship-Aware Scoring: Combines vector similarity search with Personalized PageRank to leverage the semantic relationships between entities
  • CIDOC-CRM Integration: Automatically processes RDF triples and converts ontology relationships into natural language

The system retrieves information from an RDF knowledge graph and uses network analysis to extract coherent, contextually-connected answers about cultural heritage data.
