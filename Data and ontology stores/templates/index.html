    About This Project

    This application provides an intelligent interface to explore Byzantine art and architecture data through a knowledge graph based on the CIDOC-CRM ontology. It combines:

    • RDF Data Store: A semantic database containing detailed information about Byzantine churches, frescoes, icons, and their relationships
    • CIDOC-CRM Ontology: The formal structure that organizes cultural heritage information in a standardized way
    • Vector Embeddings: Advanced AI techniques for semantic similarity search and natural language understanding
    • Retrieval-Augmented Generation: A conversational AI system that answers questions using the knowledge graph

    The system allows researchers, students, and enthusiasts to explore Byzantine art through natural language interactions, geographical context, and ontological relationships.
